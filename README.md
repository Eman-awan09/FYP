# Campus Info, Complaint & Resource Desk System

## Overview
The Campus Info, Complaint & Resource Desk System is a full-stack web application designed to digitize and centralize university operations including complaint management, resource authorization, and campus information dissemination.

The system replaces manual workflows with a secure, role-based, and efficient digital platform tailored for modern higher education environments.

This project is developed as a Final Year Project for the Department of Computer Science, Lahore Garrison University.

---

## Purpose
This system is designed to:
- Digitize complaint registration and tracking
- Provide real-time complaint monitoring and transparency
- Enable role-based dashboards for different user categories
- Automate resource request and authorization workflows
- Improve communication between students, faculty, and administration
- Provide campus information and chatbot-based assistance
- Send automated email notifications for system updates

---

## Key Features

### Student Module
- Submit and track complaints
- View campus announcements and events
- Access chatbot for academic assistance

### Teacher Module
- Approve or verify resource requests
- Submit priority complaints

### Admin Module
- Manage users, complaints, and events
- Monitor system activity and workflows
- Full system control and oversight

### Service Provider Module
- View assigned complaints
- Update complaint status and progress

### Server Room Staff Module
- Verify and process resource requests
- Maintain issuance records
